<small>Developed and tested on windows</small>

### To install tsapir globally and use cli commands

```
npm i -g tsapir
```

CLI commands:

```
h, help                                 Help: Returns CLI commands
n, new      Param: <name> Optional      New: Generate a new application
g, generate                             Generate
```
